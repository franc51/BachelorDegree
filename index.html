😀 Hello, my name is Szasz Francisc, I am student at Universitatea Transilvania, and this is my Bachelor's Degree Project. It consists of more stages which i'm going to explain down below. The project is based on an AB class amplifier( so called push-pull) which has an efficiency of maximum 78%, a preamplifier based on the NE5532 integrated circuit, custom made power-supplies, and an arduino monitoring system.

😉 In the following moments, i'm going to explain every stage, where i'm going to include all the components used, all the necesarry calculations, every program i used, how i simulated and how i used the results in order to understand and adapt to future develop.
